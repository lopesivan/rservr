Overview of the Guide.

This guide outlines basic requirements for developing C/C++ Resourcerver client programs. All client programs should follow the recommendations of this guide. This guide should also be a primary reference when designing Resourcerver systems as a whole.

This guide doesn't cover the entire Resourcerver API. Please refer to the C/C++ API Reference for a full reference of the API. The User's Guide describes abstract workings and basic execution procedures of Resourcerver systems and should also be referenced while developing clients.
